# 삼국지: 천하결전 — 탐방(뽑기) 시뮬레이터

인게임 탐방 화면을 재현한 뽑기 시뮬레이터입니다. 단일 HTML 파일로 동작합니다.

**플레이:** https://say-bot.github.io/tenha-gacha-sim/

## 기능
- 1회/5회 탐방, 전환 연출 + 카드 플립 공개(전설 서스펜스·효과음)
- 공시 확률 재현: 전설 8% / 영웅 30% / 희귀 62% (천장 포함 종합), 20회 천장
- 누적 과금액 환산 표시 (금화 1 = 옥벽 1, 옥벽 8,480개 = 149,000원)
- 획득 도감(최근순·중복 횟수), 중복 획득 시 "전환 완료" 처리
- 목표 뽑기 시뮬레이션: 원하는 무장/전법이 나올 때까지 고속 탐방 → 예상 비용 확인
- 모바일 대응
